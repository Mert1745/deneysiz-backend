# deneysiz-backend

Backend project for 'Deneysiz' mobile application, which shows brands that test their products on animals. Deneysiz also provides shop names that have these suspicious brands on sale. It is a social responsibility project and made for <a href="https://www.deneyehayir.org/">'Deneye HayÄ±r Charity'</a>.
